# Carbon4Flow – Carbon Projects Dashboard

Public dashboard for exploring REDD+ and carbon projects data.

## Features

- Interactive project map
- Vintage analysis
- Credit issuance analysis
- Storytelling section
- Dynamic KML loading per project

## Data Sources

- Google Drive (project datasets)
- Local KML files (stored in /kml directory)
- Global Forest Watch layers

## Tech Stack

- Streamlit
- Geopandas
- Folium
- Scikit-learn
- Plotly
